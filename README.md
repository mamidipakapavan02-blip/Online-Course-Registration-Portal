# 🎓 Online Course Registration Portal

A responsive Online Course Registration Portal built using HTML5, CSS3, and Bootstrap 5.

This project allows students to browse available courses, register online, learn about the institute, and contact the training center through a user-friendly interface.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- Bootstrap 5

---

## 📌 Features

### Home Page
- Responsive Navigation Bar
- Hero Section
- Featured Courses
- Student Testimonials
- Footer

### Courses Page
- Course Listing
- Bootstrap Cards
- Responsive Grid Layout
- Registration Buttons

### Registration Page
- Student Registration Form
- Resume Upload
- Terms & Conditions
- Bootstrap Alert
- Bootstrap Modal

### About Us Page
- Institute History
- Vision
- Mission
- Faculty Information

### Contact Us Page
- Contact Form
- Contact Details
- Google Map Placeholder
